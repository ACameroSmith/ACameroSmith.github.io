  4. 4. Saving the submitted information in a file on the server.
    This is one of the main reasons we are using php. To proceed, we need to learn a bit about the php functions that work with files. But there is no simple tutorial to follow, so I will guide you with the basics. We will use a few functions available in the complete reference for the php Filesystem functions.
    Search in this page for the following six functions that we will frequently use (today, only three of them: open, close and write). In each case, read and replicate the example showing how to use the corresponding php function (you can finish the other three after the class):
    • fopen: opens a file or url. Retain that you can open a file just to read it, or to write into it, or to append to it. Retain the syntax for each of these three possibilities. The example there is with the "read" option. For write, replace "r" by "w".
    • fclose: closes a file or url.
    • fread: reads from a file.
    • fwrite: writes to a file. Notice what happens if the file to which you want to write does not exist.
    • file_exists: checks if a file or folder exists.
    • fglob: returns an array with all the files and folders present in a given folder (directory).
