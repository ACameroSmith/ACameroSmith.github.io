• Each tutorial or lab exercise should be organized as a separate folder with its own html/ css/ javascript/ php files (as applicable), to make the code transparent and easy to debug. When combining several exercises, you copy the necessary code to a new renamed folder. Use your best judgement to decide if it may be necessary to group some logical components of the web app into further sub-folders, according to their functionality.
  • If you get stuck with some error that you cannot fix by the deadline: please insert a note on the main html page of the lab or homework, with a description of the bug.
  • Use the following organization for each separate web app:
    • Style files: in subfolder css.
    • Image files: in subfolder images.
    • Javascript files: in subfolder js.
    • Php files: in subfolder php.
    • Data files: in subfolder data.
    • Output files: in subfolder output.
